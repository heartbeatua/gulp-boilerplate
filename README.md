# Gulp boilerplate 

## Features

* Sass for stylesheets (with lint)
* ES6 for JavaScript
* Symbol svg-sprite

## Run

- `gulp`: Just compile files.
- `gulp --minify`: Compile files and minify them.
- `gulp dev`: Run development server.
- `gulp build --minify`: Production ready build.
